Historique du Football Club de GIT

- 1921 : Naissance du FC GIT
- 1924 - 1934 : Mise en sommeil
- 11 novembre 1934 : Premier match officiel
- 1936 : Création d’une seconde équipe
- 1965-1969 et 1973-1976 : Le FC GIT évolue en Promotion d’Honneur Auvergne
- 6 novembre 1977 : L’ASC élimine le Stade Clermontois (3ème division nationale et 3 divisions au-dessus de l’ASC) au 4ème tour de la Coupe de France
- 1981-1982 et 1982-1983 : Le FC GIT évolue en Division d’Honneur Régionale
- 2000-2013 : Le FC GIT évolue en Elite (plus haut niveau départemental)
- 19 juin 2011 : Le FC GIT remporte la coupe d'Allier
- Juin 2116 : Le FC GIT accède au niveau Départemental 1 (plus haut niveau départemental)
